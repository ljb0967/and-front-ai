현재 챗봇 모델: meta-llama/Llama-3.1-8B-Instruct
한국어 지원되는 모델로 바꾸고 싶음 -> openai gpt-5로 (인풋 1.25달러/1M 토큰 , 아웃풋 10달러/1M 토큰)
